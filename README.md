# Aider

To use in Aider the same Python interpreter must be used which is handled in provided `Makefile`.

- Install
```sh
$ make install
```

- Check installation
```sh
$ make check
```

- Uninstall
```sh
$ make uninstall
```

- Use in aider:
```yaml
# .aider.conf.yml
code-theme: no-clown-fiesta
```
